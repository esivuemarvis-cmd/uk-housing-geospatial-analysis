# 🏡 UK Housing Geospatial Analysis

This project explores UK housing prices using geospatial data analysis and interactive visualisation.

## 🚀 Features

- Data cleaning and transformation using Pandas
- Spatial analysis with GeoPandas
- Choropleth mapping of UK housing prices
- Interactive dashboard built with Streamlit

## 🛠️ Tech Stack

- Python
- Pandas
- GeoPandas
- Matplotlib
- Streamlit
